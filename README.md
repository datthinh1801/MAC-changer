# MAC changer

## Usage
```
usage: MAC_changer [-h] -i [INTERFACE] -m [NEW_MAC]

optional arguments:
  -h, --help            show this help message and exit
  -i [INTERFACE], --interface [INTERFACE]
                        the interface to be changed MAC address
  -m [NEW_MAC], --mac [NEW_MAC]
                        new mac address
```
